# HTML Advanced

Ce projet contient tous les exercices HTML avancés pour Holberton School.

## Structure du projet

- **0-index.html à 36-index.html** : Fichiers progressifs de construction de la page principale
- **index.html** : Fichier final de la page principale
- **article.html** : Page d'article
- **about.html** : Page À propos
- **latest_news.html** : Page des dernières nouvelles
- **contact.html** : Page de contact
- **11-styleguide.html à 39-styleguide.html** : Fichiers progressifs du guide de style
- **styleguide.html** : Fichier final du guide de style
- **images/** : Dossier contenant les images du projet
- **favicon.ico et favicon.png** : Icônes du site
- **logo-black.png** : Logo du site

## Description

Ce projet couvre les concepts HTML avancés incluant :
- Structure sémantique HTML5
- Métadonnées et SEO
- Navigation et liens
- Images et multimédia
- Tableaux et listes
- Formulaires (si applicable)
- Accessibilité

## Auteur

Projet réalisé dans le cadre du cursus Holberton School.
