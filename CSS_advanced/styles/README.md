# Styles Directory

This directory contains all the CSS files for the CSS Advanced project.

## File Structure

The CSS files are numbered sequentially from 1 to 32, representing progressive iterations of the stylesheet development.

### Style Files

- **1-style.css** - Initial CSS setup with custom properties
- **2-style.css** - Basic styling foundation
- **3-style.css** - Typography basics
- **4-style.css** - Root and body styling
- **5-style.css** - Container and section styling
- **6-style.css** - Grid layouts
- **7-style.css** - Navigation styling
- **8-style.css** - Header elements
- **9-style.css** - Footer styling
- **10-style.css** - Section titles
- **11-style.css** - Additional section styling
- **12-style.css** - Card components
- **13-style.css** - Services section
- **14-style.css** - Work section
- **15-style.css** - About section
- **16-style.css** - Latest news section
- **17-style.css** - Testimonials section
- **18-style.css** - Contact section
- **19-style.css** - Button styling
- **20-style.css** - Link styling and hover effects
- **21-style.css** - Interactive elements
- **22-style.css** - Form styling
- **23-style.css** - Advanced selectors
- **24-style.css** - Pseudo-classes
- **25-style.css** - Pseudo-elements
- **26-style.css** - Transitions
- **27-style.css** - Animations
- **28-style.css** - Transform properties
- **29-style.css** - Advanced layout techniques
- **30-style.css** - Responsive design basics
- **31-style.css** - Media queries
- **32-style.css** - Final polishing and optimization

## Development Approach

Each file builds upon the previous one, gradually adding more sophisticated CSS features and styling. This incremental approach helps understand how complex stylesheets are constructed step-by-step.

## Usage

Link the appropriate CSS file in your HTML document:

```html
<link rel="stylesheet" href="styles/[number]-style.css">
```

For the final complete version, use:

```html
<link rel="stylesheet" href="styles/32-style.css">
```
