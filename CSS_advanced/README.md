# CSS Advanced

This project covers advanced CSS concepts and techniques for building modern, responsive, and visually appealing websites.

## Project Structure

```
CSS_advanced/
├── images/          # High-resolution images for the project
├── styles/          # Progressive CSS stylesheets (1-32)
└── README.md        # This file
```

## Styles

The `styles/` directory contains 32 CSS files that progressively build the complete stylesheet:

- Files 1-10: Foundation (custom properties, typography, layout basics)
- Files 11-18: Component styling (sections, cards, navigation)
- Files 19-27: Interactive elements (buttons, links, transitions, animations)
- Files 28-32: Advanced techniques (transforms, responsive design, optimization)

## Learning Objectives

- Master CSS custom properties (variables)
- Implement modern layout techniques (Flexbox, Grid)
- Create smooth transitions and animations
- Apply advanced selectors and pseudo-elements
- Build responsive designs with media queries
- Follow CSS best practices and architecture

## Usage

Link the appropriate CSS file in your HTML:

```html
<link rel="stylesheet" href="styles/32-style.css">
```

Reference images:

```html
<img src="images/pic-about-01.jpg" alt="About us">
```

## Requirements

- All CSS files are validated and follow best practices
- Images are optimized for web while maintaining quality
- Code is modular and reusable
- Responsive design principles are applied
- Cross-browser compatibility ensured
